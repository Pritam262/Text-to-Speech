# Text to Speech
## Pyttsx3

Pyttsx3 is a Python library for dealing Text to speech convert.

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install pyttsx3.

```bash
pip install pyttsx3
```
